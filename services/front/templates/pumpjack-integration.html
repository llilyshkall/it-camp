Управление анимацией:

  • Start/Pause - запуск/остановка анимации
  • Reset - сброс в начальное положение
  • Speed - настройка скорости (0.3 - 1.2 циклов/сек)
  • Stroke - настройка хода качалки (10° - 22°)
  • Show pivots - показ точек поворота

Технические характеристики

Компонент включает в себя:

  • Изолированные CSS стили
  • JavaScript API для управления
